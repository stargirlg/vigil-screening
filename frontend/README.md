# VIGIL Frontend

Frontend application for the VIGIL AML Compliance Screening Platform.

Built with React 19, TypeScript, and Vite, the application provides an intuitive interface for customer screening, alert management, case investigation, analytics, and compliance workflows.

---

## Tech Stack

- React 19
- TypeScript 5
- Vite
- Tailwind CSS v4
- Axios
- Recharts
- Tabler Icons

---

## Features

- JWT-based authentication
- Dashboard with AML compliance metrics
- Customer screening interface
- Alert management
- Case management
- Analytics dashboards
- Responsive user interface

---

## Prerequisites

- Node.js 20 or later
- npm

---

## Installation

```bash
npm install
npm run dev
```

---

## Production Build

```bash
npm run build
```
